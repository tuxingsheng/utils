# util - 常用工具函数集合
```

isArray             验证是否是数组
isString            验证是否是字符串
hasOwn              验证该属性是否属于某个对象
isObject            验证是否是对象
isEmpty             是否为空，支持检测数组、对象和字符串
extend              对象拷贝
each                each循环
trim                去掉左右空格
resolveRootSearch   解析客户端location search参数
device              获取浏览器信息
formatDate          时间戳转换为字符串时间格式
